Steps to update the firmware on the MHD Ethanol Content Analyzer

1. install the CH34x driver located in the /driver folder
2. plug a USB cable into your PC and the ECA
3. double click XLoader.exe
4. browse to find the correct *.hex file for firmware
5. ensure device is MHD ECA v1.0
6. correct COM port can be found in Device Manager, under "Ports"
7. ensure Baud Rate is 57600
8. click Upload
9. message should appear under Upload button indicating the number of bytes written

Success!


Please note: this firmware is only to be used with the official MHD Ethanol Content Analyzer (ECA).
Other usage is strictly prohibited.
