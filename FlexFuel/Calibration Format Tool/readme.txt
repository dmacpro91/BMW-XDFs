Steps to convert a regular bin to a Flex Fuel bin:

1. drag and drop your current *.bin tune onto FlexFuelMapBuilder executable
2. press any key to close the exe
3. you will see a new *_FF.bin file in the same directory as the one you drag and dropped

Success!

Please note: The output of this executable is intended to be flashed via MHD only.
Using a different tool could cause vehicle not to start or other negative side effects.
